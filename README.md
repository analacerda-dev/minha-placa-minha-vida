#  Minha Placa Minha Vida

Landing page e sistema de diagnóstico online gratuito para assistência técnica especializada em reparo de placas eletrônicas e micro-soldagem.

O projeto foi desenvolvido com foco em performance, experiência do usuário e conversão via WhatsApp.
---

##  Funcionalidades

- Landing page institucional
- Página de Serviços
- Página de Diagnóstico Online Gratuito
- Formulário integrado ao WhatsApp
- Layout responsivo (desktop e mobile)
- Componentização reutilizável
- Design consistente

---

## Tecnologias Utilizadas

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **React Router DOM**
- **Lucide React (ícones)**

---
## Como rodar o projeto localmente

### 1. Clone o repositório
```bash
git clone https://github.com/analacerda-dev/minha-placa-minha-vida
```
### 2. Instale as dependências
```bash
npm install
```
### 3. Inicie o servidor de desenvolvimento
```bash
npm run dev
```
